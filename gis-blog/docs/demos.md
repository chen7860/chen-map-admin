# Demo 索引

这些链接默认指向 `chen-map-admin` 主应用中的示例路由。

| 功能 | 主应用路径 | 说明 |
| --- | --- | --- |
| Cesium 雨雪天气 | `/cesium/rain` | 后处理与粒子效果 |
| Cesium 飞线轨迹 | `/cesium/flow-line` | 中心点向周边节点的动态流动线 |
| Cesium 自定义材质 | `/cesium/material` | 绘制流动线材质 |
| Leaflet GeoJSON | `/leaflet/geo-json-render` | 点线面数据渲染 |
| Leaflet 地图交互 | `/leaflet/map-interaction` | 坐标拾取、框选、右键菜单 |

后续可以把 Demo 做成可配置的外链，比如开发环境指向 `http://localhost:8848/cesium/rain`，生产环境指向线上地址。

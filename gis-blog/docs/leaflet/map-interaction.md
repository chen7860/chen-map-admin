# Leaflet 地图交互增强

## 功能简介

地图交互增强通常是业务 GIS 系统最常用的一类能力，包括坐标拾取、右键菜单、框选点位、视图重置等。

这些能力看起来基础，但对业务系统很重要，因为用户需要在地图上完成选择、定位、复制、标记等操作。

## 功能拆分

- 鼠标坐标实时显示。
- 点击地图添加临时标记。
- 右键菜单复制坐标或飞到当前位置。
- 框选范围内的点位。
- 清空选择和重置视图。

## 实现建议

1. 地图事件绑定和解绑要成对出现，避免页面切换后残留。
2. 框选时临时关闭地图拖拽，避免拖动地图和画框冲突。
3. 坐标格式要统一，比如 `lng, lat`。
4. 业务点位选中状态应该由数据驱动，而不是只改图层样式。

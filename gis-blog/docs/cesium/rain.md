# Cesium 雨雪天气效果

<a class="demo-link" href="/cesium/rain" target="_blank">打开主应用 Demo：/cesium/rain</a>

## 功能简介

雨雪天气适合用来增强三维地图的现场感，比如应急指挥、交通态势、园区巡检、城市运行监控等场景。

当前示例主要由两部分组成：

- 后处理 shader：给画面增加阴天、湿润、雪花或风线效果。
- 粒子系统：在雨天模式下生成更明显的前景雨线。

## 适用场景

<ul class="meta-list">
  <li>应急指挥：模拟恶劣天气下的城市运行环境。</li>
  <li>交通管理：展示降雨、降雪对道路通行的影响。</li>
  <li>可视化演示：让地图 Demo 更容易被业务用户理解。</li>
</ul>

## 实现思路

1. 使用 `PostProcessStage` 对最终画面做统一调色。
2. 雨天叠加 `ParticleSystem`，让近景雨线更明显。
3. 不同天气使用不同配置：密度、速度、倾斜角、透明度。
4. 雪花效果用 shader 生成多层不同尺度的随机雪片。

## 常见问题

- 背景太亮时，雨线和雪花会不明显，需要降低天空亮度或切换到城市近景。
- 过高的密度会让画面变脏，应该优先调大粒径，再微调密度。
- 后处理应当支持关闭和恢复场景状态，避免切换页面后影响其他 Cesium 页面。
